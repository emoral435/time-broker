## From Source

```shell
go install github.com/emoral435/time-broker@latest
```

## From a Release

Download the latest binary from the
[releases page](https://github.com/emoral435/time-broker/releases) for your
platform:

- macOS (arm64 / amd64)
- Linux (arm64 / amd64)
- Windows (arm64 / amd64)

## Using curl (future)

```shell
curl -fsSL https://github.com/emoral435/time-broker/install.sh | sh
```
